# Dream Duels: クイックリファレンス

> 全ルール: [rules.md](./rules.md) (EN) · [rules.ja.md](./rules.ja.md) (JA)

## 勝利条件

最後まで生き残ったチームの勝利。チームのメンバー全員のライフトークンが 0 に
なった時点でそのチームは敗北。

## フェーズ順序 (毎ラウンド)

1. **バトル** — エンカウント・カード開示・ダメージ
2. **禁止エリア** — GM が山から 2 枚引いて新禁止エリアを決定
3. **移動** — GM が山から 1 枚引く (移動属性) → チームが移動 or 回転
4. **復活** — 自チームの座標に落ちているライフトークンを回収

## 三すくみ

```text
火 → 木 → 水 → 火 (循環)
ジョーカー: 全属性に勝つ
同属性: あいこ
```

| A が出す | B が出す | 結果 |
| --- | --- | --- |
| 火 | 木 | A の勝ち |
| 木 | 水 | A の勝ち |
| 水 | 火 | A の勝ち |
| 火 | 火 | あいこ |
| — | — | カード不備: カードを出した方が勝ち |
| ジョーカー | 何でも | ジョーカーの勝ち |
| ジョーカー | ジョーカー | あいこ |

## ダメージポイント

| 条件 | ポイント |
| --- | --- |
| 基本 (三すくみで勝利) | 1 |
| 向きボーナス: 直交していない + 同一座標 | +1 |
| 向きボーナス: 直交していない + 相手に向いている + 隣接座標 | +1 |
| ボーナス: 自分の数値 ≥ 相手の 2 倍 | +1 |
| ボーナス: 自分が 13、相手が 7〜10 | +1 |
| ボーナス: 自分が 2 以下、相手が 11 以上 | +1 |
| ペナルティ: ボーナスの逆条件 | −1 |
| ジョーカー勝ち・カード不備勝ち | 固定 1 (上記全てスキップ) |
| あいこ | 双方 0 |

ダメージは 0 未満にはならない。

## ライフトークンの上限

- 1 人あたりの上限: **4** (ソロチームの場合 **3** でスタート)
- ダメージを受けたトークンは自チームの座標にドロップ
- 復活フェーズ: 条件を満たせばドロップされたトークンを 1 つ回収可能
- ライフチャージ上限: 4

## 移動ルール

- **カードが移動属性と一致** → 現在の向きに 1 マス進む
- **カードが移動属性と不一致** → 提示したカードの向きにコマを回転させる
- **グリッド外 or 禁止エリア** → ライフトークン 1 を清算 (自滅のためカード強奪なし)

## 座標系

3×3 グリッドの各軸は火・水・木。
コマ = **番号トークン** (チーム ID) + **属性カード 2 枚**
(カードの向き = チームの向き)。

## カードの種類

| カード | 特殊ルール |
| --- | --- |
| 属性カード (1〜13) | 通常プレイ; 数値によりボーナス・ペナルティあり |
| ジョーカー | 全属性に勝つ; ダメージ固定 1; エンカウント順では番号 25 扱い |
